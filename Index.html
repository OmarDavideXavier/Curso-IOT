Projecto de Controle Remoto de Iluminação de Led


Omar Davide Xavier, estudante da faculdade de engenharia, UEM é o criador deste interessante projeto de controle de iluminação de LED. Através de uma página web interativa, é possível selecionar uma imagem/botão ou um LED específico para ligar ou desligar a lâmpada correspondente. Com uma abordagem intuitiva e fácil de usar, esse sistema permite controlar a iluminação de forma personalizada e eficiente. Uma solução inovadora que combina tecnologia e praticidade para criar uma experiência única de controle de iluminação. TME Education Mozambique
